# bomba

## How to run:
First you'll need to clone the repo to your local machine, to run the FRONTEND you need to install [Node](https://nodejs.org/en/)
### Clone Repo:

```bash
  $ git clone https://github.com/gustavobaltazar/bomba.git
  $ cd bomba
```

### On client:
```js
  $ cd client
  $ pnpm i
  $ pnpm dev
```

### On server:
```js
  $ cd client
  $ pnpm i
  $ cd src
  $ node app.js
```
